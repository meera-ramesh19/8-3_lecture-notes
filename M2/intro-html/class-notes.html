Review Questions

  • What does HTML stand for?
    • Hyper Text Markup Language
    • Defines WHAT/the structure a browser user sees on a page
    • Hyper Text: text documents that are linked to other text documents
  • What does it mean that HTML is a markup language? How is this different than JavaScript?
    • Markup Language: Computer language that uses tags to define elements, end result is meant to be human readable
    • Markup language: we markup human readable content with info on how it should be displayed
    • Difference from JS: JS is not meant to be human readable, it’s a programming lang for computers to read
    • JS: the HOW web page behaves
  • What is an HTML element?
    • Everything from a start tag to an end tag
    • It tells a browser how to interpret content on a page
  • What is the difference between an HTML tag and an HTML element?
    • Tags are the symbols used to denote the beginning and end of an element
  • The following is a paragraph element. How can you tell the difference between the opening and closing tags?
    <p>
  The notable 19th-century architect of skyscrapers, Louis Sullivan, promoted
  an overriding precept to architectural design: "Form follows function".
</p>

    • Closing tags start with a </
    • Open tags don’t have the slash

Types of HTML Tags

Find your name on the list below. Next to your name is a number, compute your number modulo 4. This will correspond to which html tag you look up.

Take 4 minutes to answer ONE OF the following questions. You may find MDN’s HTML Elements documentation helpful:

  1. What is the purpose of the DOCTYPE element?
    1. Tells us what version of HTML is being used
    1. There’s more than one version of HTML!
    1. We will typically use HTML5
  1. What is the purpose of the html element?
    1. Root element for all an html doc
    1. All other elements should be descendants of the html element
    1. There should only be ONE
  1. What is the purpose of the head element?
    1. contains metadata for a document
    1. it’s not displayed
    1. what is metadata? It’s information ABOUT a page that shouldn’t be displayed ON the page.
    1. examples: document title, language used, scripts (much more on this later), etc
    1. There should only be ONE
  1. What is the purpose of the body element?
    1. Represents the content of the HTML Doc
    1. There can only be ONE
    1. Everything a use sees, will be contained in the body

CSS in VS Code

  • VS Code will suggest relevant properties AND values - use this to your advantage!
  • The point of class is NOT to cover every css property out there. You will naturally memorize many that you use frequently. You will have to look up CSS properties, their meanings, and example uses frequently.
  • Good way to practice/study css: try to clone website designs you like
  • Bad way to practice/study css: memorizing common css properties and their meanings

CSS Lab Workflow, Plus Tangent

  • How is “debugging” css different from debugging JavaScript?
  • Is CSS a programming language?
    • rabbit hole warning!
    • CS folks call machines/programming languages that can handle sufficiently complex computational tasks “turing complete”
    • CSS and HTML are technically turing complete
    • This is evidenced by an implementation of rule 110 in css and html
    • Actually writing css/html that could, for example, add two numbers is pretty impractical
    • The point is, it’s blurry exactly what counts as a programming language, and css lies somewhere within that blurry border
